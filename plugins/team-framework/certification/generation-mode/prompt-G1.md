## 依頼者の仕様（原文）

### ポイント付与機能

- 購入確定時、税抜購入額の 3% をポイントとして付与する。
- 入力は税抜金額（正の整数・円）。エラー処理・入力バリデーションは呼び出し側の責務であり本仕様の範囲外。性能要件なし。

### 受け入れ例（依頼者提供。実装はこの表と完全一致すること）

| 税抜購入額 | 付与ポイント |
| ---------- | ------------ |
| 1,000 円   | 30 pt        |
| 1,050 円   | 32 pt        |
| 2,400 円   | 72 pt        |
