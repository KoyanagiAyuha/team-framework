## 依頼者の仕様（原文）

### 動画配信サブスク：解約時の返金額計算

#### 背景

月額制の動画配信サービス。解約は即時に成立し、解約日の翌日から視聴できなくなる（解約日当日は利用日として扱う）。カスタマーサポートは利用者に「1日あたりの料金 × 残りの日数を日割りでご返金します」と案内している。

#### 規則

- 返金額は、月額料金を解約月の実日数で日割りし、未利用日数を乗じた額の 1 円未満を切り捨てた金額とする。
- 未利用日数は、解約月の実日数から解約日の日付（その日を含む経過日数）を引いた日数とする。
- 月額料金は正の整数（円）。解約日は `YYYY-MM-DD` 形式。タイムゾーンは考慮不要（日付文字列のまま演算してよい）。
- 関数シグネチャ: `cancellationRefund(monthlyFee: number, cancelDate: string): number`（返金額・円・整数）
- エラー処理・入力バリデーションは呼び出し側の責務であり本仕様の範囲外。性能要件なし。

### 計算例（経理部確認済み）

| 月額     | 解約日     | 返金額   |
| -------- | ---------- | -------- |
| 3,000 円 | 2026-06-20 | 1,000 円 |
